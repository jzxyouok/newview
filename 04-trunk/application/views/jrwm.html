客户经理

招聘人数：2名

职位描述：
1. 开发公司新客户及维护客户关系。
2. 独立负责公司签约客户，日常沟通。
3. 能够独立为客户制定企业宣传方案。
4. 高质量的完成项目中各个方面工作。
5. 能独立完成项目执行过程中的沟通协调。
任职要求：
1. 本科及以上学历，市场专业优先考虑。
2. 2年以上客户经理经验，广告公司就职经验者优先考虑。
3. 项目控制能力与良好的客户沟通能力。
4. 具有一定的团队管理经验。

工作地点：北京

文案策划

招聘人数：2名

职位描述：
1. 撰写公司各项目方案的策划。
2. 参与项目的谈判和对客户进行提案。
3. 知识面广泛，文笔佳，思路清晰。
4. 良好的沟通能力及团队合作精神
5. 具有品牌及产品的分析能力,为品牌创造最佳价值
任职要求：
1. 本科及以上学历，2年以上创意策划经验。
2. 精通办公自动化软件，尤其能娴熟地进行PowerPoint的运用。
3. 活跃又成熟的创意思路，杰出的文字表达能力。
4. 逻辑思维能力较强。

工作地点：北京

影视后期制作

招聘人数：2名

职位描述：
1. 负责宣传片、广告等二维，三维视频特效的设计制作；
2. 参与项目的素材收集，充实公司现有资料库；
3. 参与后期制作流程，独立完成本职工作；
4. 与导演、剪辑师沟通协作，配合团队同仁完成效果表达；
5. 收集分享优秀的影视、绘画、游戏等与视觉相关的事物。
任职要求：
1. 本科及以上学历，市场专业优先考虑。
2. 2年以上客户经理经验，广告公司就职经验者优先考虑。
3. 项目控制能力与良好的客户沟通能力。
4. 具有一定的团队管理经验。

工作地点：北京
